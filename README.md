# FableRace

Браузерная 3D-гонка на Three.js: аркадная физика, замкнутая трасса, таймер кругов с секторами, мини-карта и камера-погоня.

**▶ Играть: https://n1ghtk1n.github.io/fablerace/**

## Запуск

```bash
npm install
npm run dev      # открыть http://localhost:5173
npm run build    # прод-сборка в dist/
npm run deploy   # сборка + публикация на GitHub Pages (ветка gh-pages)
```

Пути к ассетам проходят через `src/asset.js` (`import.meta.env.BASE_URL`), поэтому игра работает и в корне домена (dev), и под подуровнем `/fablerace/` на Pages — `base` задаётся в [vite.config.js](vite.config.js).

## Управление

| Клавиши | Действие |
|---|---|
| `W` / `↑` | Газ |
| `S` / `↓` | Тормоз / задний ход |
| `A` `D` / `←` `→` | Руль |
| `Space` | Ручник (дрифт) |
| `Shift` | Нитро (запас копится дрифтом) |
| `R` | Вернуть машину на трассу |
| `Enter` | Рестарт заезда |
| `Esc` | Выход в меню |
| `M` | Вкл/выкл звук |

## Как устроено

- **Трансмиссия** ([src/car.js](src/car.js)) — 6-ступенчатая АКПП: обороты считаются от скорости колёс через передаточные числа, тяга — по кривой крутящего момента, при переключении тяга разрывается на ~0.4 с (переключения реально чувствуются). Задний ход — отдельный режим (S на месте). Разгон плавный: ~0–65 км/ч за 2.6 с на первой передаче.
- **Руль** — позиция руля плавно набирается и возвращается (не мгновенный поворот), а достижимый угол колёс ограничен сцеплением шин: `lat_accel = v²·tan(угол)/база ≤ ~1g`. На 30 км/ч можно вывернуть на 20°, на 120 км/ч — только ~1.3°, поэтому резких рывков на скорости нет.
- **Дрифт** — срыв в занос кик-даун ручником (`Space` + руль на скорости) или естественно при большом угле скольжения; в заносе задняя ось «свободнее» (низкое боковое демпфирование), у руля повышенная авторитетность для контрруления, а часть погашенной боковой скорости возвращается в продольную («момент дрифта», угасает с косинусом угла — боком на 90° тяги нет). Дым из-под задних колёс ([src/smoke.js](src/smoke.js)) и живой счётчик очков дрифта в HUD. Выход из заноса — гистерезис по углу скольжения.
- **Нитро** — `Shift`: +7.5 м/с² тяги (отсечка ~230 км/ч), бак 100% выгорает за ~3 с; запас копится дрифтом (14%/с в заносе) плюс медленный пассивный прирост. Голубая дуга на спидометре, пламя из выхлопа, расширение FOV и «реактивный» свист. Боты со сложности «Гонщик» тоже дрифтуют в крутых поворотах и жгут нитро на прямых — круги ботов: Новичок ~42 с, Гонщик ~39 с, Профи ~35 с.
- **Настройки машины** ([src/settings.js](src/settings.js)) — в меню: ползунки мощности, аэродинамики (макс. скорость), остроты руля и сцепления шин; встроенные пресеты (Стандарт, Новичок, Спорт, Дрифт) и свои — сохраняются в localStorage и применяются сразу.
- **Режимы** — «Тренировка» (свободные круги с призраком) и «Гонка» (3 круга против трёх ботов). Боты ([src/ai.js](src/ai.js)) ездят на той же физике `Car` с pursuit-ИИ; сложность = дальность взгляда, поздность торможения, доля газа и сцепление: Нико (новичок, спорт-седан), Кай (гонщик, хот-хэтч), Вера (профи, прототип). Стартовая решётка со смещёнными полосами, аркадные коллизии машин, позиция и круги в HUD, соперники на мини-карте, таблица результатов на финише.
- **Призрак лучшего круга** ([src/ghost.js](src/ghost.js)) — траектория пишется каждый круг (~60 Гц: время, позиция, курс); круг, ставший лучшим, становится призраком — полупрозрачной копией машины без коллизий, которая реплеится с интерполяцией по времени и стартует с линии на каждом твоём круге. Медленные круги призрака не трогают; если призрак финиширует раньше — ждёт у линии.
- **Звук** ([src/audio.js](src/audio.js)) — двигатель синтезируется Web Audio без сэмплов: четыре слоя осцилляторов (такты зажигания, коленвал, выхлоп, клапана) с частотой от оборотов, лоупасс-фильтр открывается под нагрузкой, при переключении передачи — провал тяги и скачок тона; плюс шум ветра от скорости. По умолчанию звук **выключен**; `M` включает, выбор запоминается в localStorage. В фоне вкладки звук глушится.
- **Гараж** ([src/cars.js](src/cars.js)) — пять машин с разными сбалансированными профилями (болид, прототип, спорт-седан, хот-хэтч, карт): профиль перемножается с ползунками настроек, у карта короче база и меньше размер. Выбор — карусель в меню с полосками характеристик, сохраняется в localStorage; по умолчанию — спорт-седан.
- **Поверхности** — на траве падает тяга (буксование) и растёт сопротивление (плавно от скорости, поэтому тронуться с места можно всегда): больше ~35 км/ч по траве не разгонишься. Удар о борт гасит скорость пропорционально «прямоте» удара, но только один раз за контакт; при продолжающемся касании гасится лишь составляющая в сторону стены — вдоль борта можно скользить, и задним ходом всегда можно уехать.
- **Трек** ([src/track.js](src/track.js)) — замкнутый Catmull-Rom-сплайн; дорога и трава — фототекстуры ambientCG с нормалями, разметка рисуется поверх асфальта на canvas. По обеим сторонам — красно-белые борта (физическая граница мира). Деревья — InstancedMesh с детерминированным PRNG.
- **Тайминг** ([src/timing.js](src/timing.js)) — круг засчитывается только после прохождения трёх секторов по порядку (срезать или пересечь финиш задним ходом не выйдет).
- **HUD** ([src/hud.js](src/hud.js)) — аналоговый спидометр (стрелка, шкала до 240), дуга тахометра с красной зоной, в центре текущая передача (или R) и цифра скорости под ней; мини-карта.
- **Физика** идёт с фиксированным шагом 120 Гц независимо от FPS ([src/main.js](src/main.js)); игра начинается со стартового меню, Esc возвращает в него.
- **Графика** — атмосферное небо (three.js `Sky`) запечено в environment-карту (IBL-отражения), ACES-тонемаппинг, низкое тёплое солнце с мягкими тенями 3К, кузов — MeshPhysicalMaterial с клиркоутом поверх усиленной карты цветов, постобработка через [postprocessing](https://github.com/pmndrs/postprocessing): bloom, виньетка, цветокоррекция, MSAA ×4. Холмы на горизонте, два вида деревьев с вариацией оттенков, рекламные щиты вдоль прямых.

## Ассеты

- Модель машины (`race.glb`) и конусы — [Kenney Car Kit](https://kenney.nl/assets/car-kit), **CC0** (см. [public/models/License.txt](public/models/License.txt)).
- Текстуры асфальта и травы — [ambientCG](https://ambientcg.com) (Asphalt025C, Grass004), **CC0** (см. [public/textures/CREDITS.txt](public/textures/CREDITS.txt)).

## Идеи для расширения

- Звук двигателя (Web Audio, тон от оборотов).
- Призрак лучшего круга (записывать позиции, проигрывать полупрозрачной машиной).
- Несколько трасс — достаточно поменять `CONTROL_POINTS` в `track.js`.
- Соперники-боты: pursuit-автопилот по центральной линии уже проверен в разработке.
- Геймпад через Gamepad API (аналоговый газ/руль).
